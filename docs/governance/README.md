# Governance

## Zakres
- Strategia zarządzania repozytorium i zespołem.
- Dokumenty typu SLA, kontrakty repo, polityki i cele.
- Raporty z przeglądów, usprawnień i decyzji architektonicznych wysokiego szczebla.

## Zadania startowe
1. Zweryfikować przeniesioną z `docs/governance/` zawartość (frontmatter, linki, folder notes).
2. Zaktualizować odnośniki w indeksach (`dev/INDEX.md`, `docs/FULL-INDEX.md`, `docs/infra/RESTRUCTURE-MAP.md`).
3. Powiązać dokumenty z katalogiem `business/processes/` (procesy operacyjne) i `business/vendors/` (umowy współpracy).
