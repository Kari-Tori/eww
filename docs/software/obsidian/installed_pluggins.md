Poniższa lista została wyekstrahowana z frontmatter szablonu dziennego (obsidian-kanban mockup). Krótkie opisy wskazują typowe zastosowanie.

1. Kanban — tablice Kanban w notatkach (zarządzanie zadaniami na tablicy)
2. Dataview — zapytania i raporty z notatek (raportowanie, widoki danych)
3. Templater — skrypty i dynamiczne szablony (automatyzacja szablonów)
4. Tasks — zaawansowane filtrowanie zadań (filtry, due, priorytety)
5. QuickAdd — szybkie akcje / makra (szybkie tworzenie kart i notatek)
6. Calendar — kalendarz (integracja z daily/periodic notes)
7. Periodic Notes — notatki periodyczne (daily, weekly, monthly)
8. Obsidian Git — backup i automatyczne commity (wersjonowanie vaulta)

Uwagi:
- Użyj tego pliku jako referencji przy instalacji wtyczek w vault.
- Dodatkowe konfiguracje i przykłady znajdują się w: /docs/software/obsidian/plugins.md
