# Methodologies

## Zakres
- Opisy metodyk (PARA, Blitzscaling, DevOps), przewodniki pracy i frameworki decyzyjne.
- Materiały edukacyjne dotyczące procesów biznesowych lub PMO.

## Zadania startowe
1. Zweryfikować przeniesione pliki z dawnego `docs/methodologies/` oraz zdecydować, które przewodniki z `docs/infra/guide/` powinny trafić do warstwy biznesowej.
2. Ustalić standard sekcji (cel, zasady, kroki wdrożenia) dla każdej metodyki.
3. Dodać indeks linkujący do powiązanych procesów w `business/processes/`.
