# Odoo – narzędzie operacyjne

Odoo służy do:
- Zarządzania inwentarzem i przyjęciami (INTAKE)
- Obsługi zleceń napraw (REPAIR)
- Śledzenia statusów produktów i transakcji (SALES)
- Automatyzacji procesów biznesowych

Pliki integracyjne, skrypty i dokumentacja Odoo umieszczamy w tym folderze.
