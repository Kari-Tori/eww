Import vault: lokalna kopia surowego vaulta

Ten katalog zawiera lokalnie przeniesioną, surową kopię vaulta Kariny.

- `karinam-vault/` — pełna kopia importu (lokalna). Pliki w tym katalogu są ignorowane przez Git, aby uniknąć przypadkowego dodania dużych danych.
- Aby zachować pełny surowy import w repo, sprawdź gałąź `archive/imported-raw`.

Jeśli chcesz, mogę dodać skrypt do eksportu/archiwizacji lub przygotować plan usunięcia dużych plików z historii.
